### Project Description

LastKey is an access continuity protocol on Etherlink. A user writes a continuity plan in plain English, the app converts it into validated beneficiary data, and an onchain vault enforces the rule after a long inactivity window. A monitoring agent checks vault state, records warning milestones, and can execute the final distribution on Etherlink when the owner does not check in.
